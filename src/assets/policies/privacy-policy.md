# Privacy Policy - RelaOne
Terakhir diperbarui: 14 November 2025

Aplikasi **Volunteer Activity Manager** (â€œKamiâ€) menghargai privasi Anda dan berkomitmen melindungi data pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi data Anda saat menggunakan layanan kami.

---

## 1. Data yang Kami Kumpulkan

Kami mengumpulkan data yang Anda berikan secara langsung saat membuat akun maupun saat menggunakan fitur layanan.

---

### 1.1 Informasi untuk Pengguna Volunteer
- Nama lengkap  
- Email  
- Nomor telepon  
- Tanggal lahir  
- Jenis kelamin  
- Alamat (manual, tidak ada koordinat)  
- Password terenkripsi  
- Foto profil (opsional)  
- Bio dan keahlian (opsional)

> Tidak ada pelacakan lokasi volunteer. Semua alamat dimasukkan manual oleh pengguna.

---

### 1.2 Informasi Tambahan untuk Pengguna Organisasi
Jika Anda mendaftar sebagai **Organisasi**, kami juga mengumpulkan:

- **Nama organisasi**  
- **Deskripsi organisasi**  
- **Website organisasi (company profile)**  
- **Email organisasi**  
- **Nomor telepon organisasi**  
- **Alamat organisasi**  
- **Logo organisasi** (jika diunggah)  
- Status verifikasi organisasi  
- Statistik organisasi (jumlah event, rating, dsb.)

> Data ini digunakan untuk menampilkan profil organisasi kepada volunteer dan untuk keperluan transparansi event.

---

### 1.3 Data Event & Aktivitas
Kami mencatat:
- Event yang Anda buat (jika Anda organisasi)  
- Event yang Anda ikuti (jika Anda volunteer)  
- Proses pendaftaran event  
- Status konfirmasi & kehadiran  
- Feedback & rating  

---

### 1.4 Data Lokasi Event
Untuk event yang dibuat oleh organisasi:

- Alamat event (manual)  
- Latitude & longitude event (jika diisi)  

> Koordinat event berasal dari input manual pengguna (misalnya dari link Maps yang ditempel).  
> Kami **tidak menggunakan API Google Maps** dan tidak mengambil lokasi otomatis.

---

### 1.5 Data Autentikasi
- Token login (Laravel Sanctum API + JWT)  
- Tidak ada cookies tracking  
- Password disimpan dengan enkripsi bcrypt

---

### 1.6 Informasi Teknis
- IP Address  
- User Agent  
- Log server dasar untuk keamanan

---

## 2. Bagaimana Kami Menggunakan Data Anda

Kami menggunakan data untuk:
- Membuat dan mengelola akun volunteer atau organisasi  
- Menyimpan dan menampilkan profil organisasi  
- Mengelola event, pendaftaran, dan aktivitas pengguna  
- Autentikasi login berbasis token  
- Mengirim email verifikasi, reset password, dan notifikasi  
- Menjaga keamanan layanan  
- Meningkatkan pengalaman pengguna  

---

## 3. Pengiriman Email

Kami menggunakan **SMTP Gmail pribadi** untuk:
- Verifikasi email  
- Reset password  
- Notifikasi penting  

Tidak ada data lain yang dikirim ke Google selain alamat email tujuan & isi pesan.

---

## 4. Tidak Ada Cookies Tracking

Kami:
- Tidak menggunakan cookies tracking  
- Tidak menggunakan Google Analytics  
- Tidak menggunakan pixel pihak ketiga  
- Menggunakan **Bearer Token** untuk autentikasi

---

## 5. Keamanan Data

Kami melindungi data Anda dengan:
- Enkripsi password  
- Validasi data  
- Pembatasan role akses  
- Firewall & proteksi server  

Namun kami tidak dapat menjamin keamanan 100% dari risiko eksternal.

---

## 6. Berbagi Data dengan Pihak Ketiga

Kami **tidak menjual atau membagikan** data Anda.  
Hanya satu pihak luar yang menerima sebagian data:

- **Gmail SMTP** untuk pengiriman email sistem.

Tidak ada pihak ketiga lainnya.

---

## 7. Hak Pengguna

Anda berhak:
- Mengakses data pribadi  
- Memperbarui data  
- Menghapus akun  
- Meminta penghapusan permanen data Anda  

---

## 8. Penghapusan Data

Jika akun Anda dihapus:
- Data pribadi dapat dihapus permanen  
- Data event & feedback dapat disimpan dalam bentuk anonim untuk keperluan statistik

---

## 9. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini sewaktu-waktu.
